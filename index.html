What Are Semantic Elements?

Semantic and Non-Semantic Elements

Semantic elements, also known as HTML5 semantic tags, define the function and category of text which simplify the readability for web browsers and search engines. Using a set of globally accepted names for structural elements, search engines are more easily able to index web page content correctly and read information at a faster speed. These tags identify and define the purpose of the element, which allow browsers to understand the meaning of the content, rather than just displaying it.

Examples of Semantic Elements

  • Tables
  • Forms
  • Articles

Semantic elements are not to be confused with non-semantic elements, which don't define content. The two, most common examples of non-semantic elements include:

  1. div tags
  2. span tags
